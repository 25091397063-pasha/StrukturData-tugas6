# 🌀 Queue Insight: Visualisasi Struktur Data

> **Data Structures & Algorithms** • Eksplorasi mekanisme Queue melalui 5 simulasi interaktif berbasis Python.

---

## 💡 Overview
Repositori ini dirancang sebagai instrumen pembelajaran visual untuk memahami struktur data **Queue (Antrian)**. Menggunakan prinsip **FIFO (First In, First Out)**, setiap modul dalam proyek ini menghasilkan animasi GIF yang membedah bagaimana data mengalir dalam sistem nyata.

---

## 📂 Arsitektur Proyek
Sistem diorganisir ke dalam modul-modul mandiri untuk memudahkan eksperimen:

```text
queue-visualization/
├── case1_printer_queue.py   # Logika FIFO Dasar
├── case2_hot_potato.py      # Simulasi Sirkular (Rotasi)
├── case3_hospital_queue.py  # Skala Prioritas (Priority Queue)
├── case4_bfs.py             # Engine Penelusuran Graf
├── case5_airport_sim.py     # Simulasi Peristiwa Diskrit
├── outputs/                 # Koleksi Visualisasi (.gif)
└── README.md
